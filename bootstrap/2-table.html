Autres Types de tables

Il existe d'autres classe de type table:

Type table class Explication
Table simple .table Cree un table simple avec des bordures sur les lignes
Table Simple colorée .table et tr class="couleur" Cree un table simple avec des bordures sur les lignes et des backgrouds colorés
Table bordered .table-bordered Cree un table avec des bordures sur les lignes et collonnes
Table colore .table-dark Cree un table avec une couleur precise Ex dark
Table table-hover .table-hover Cree une interraction lors du passage sur une ligne d'un tableau
Table borderless .table-borderless Cree un table sans bordures
Table entete Colore thead class="thead-dark" Cree un tableau avec entete coloré Ex dark
Table réduit .table-sm Cree un tableau en coupant le padding des cellules sur deux
Light table-light Ligne sans bordure
Dark table-dark bordure de la ligne en noir
